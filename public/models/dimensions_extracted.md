## 関節仕様（VIMO Classic / Franz × 岡村 2026-05-22 確定）

**設計の正典**は **失効済み特許 US6893318B2**（Tech Rider FLEXDOLL / Søviknes-Sætveit / 出願2001 / 公開2005-05-17 / 失効2017-05-17 US維持費未納）。
14関節 Dummy-13 構成は廃案。VIMO Classic では特許構造をそのまま採用し、可動部は **3点だけ**にする。

| 部位 | 仕様 |
|---|---|
| 胴体（torso + 肩 + 大腿 一体） | 弾性エラストマー（シリコン or TPE）／patent Fig.2-3 |
| 上腕・前腕／大腿・下腿 | 硬質プラスチック（PA / POM / PVC）／patent Fig.6-12 |
| 肘・膝 | **予め riding stance 角度に固定**、背面の弾性腱（item 48）で保持／patent Fig.5-7 |
| 首（新規追加） | **硬質プラの swivel joint**、360°回転＋±10°チルト |
| 手首 × 2（新規追加） | **硬質プラの swivel joint**、自由回転（ボード掴み・地面押さえ用） |
| 顔・手 | シリコン剥き出し（タッチ＆グリップ感） |
| 衣装 | 競技別の本物の布。Snowboard SS26 はパファー上着＋カーゴ |

### 14関節 Dummy-13 案から VIMO Classic への切替理由

- Riding stance で自立 → 購入者にポージング技術不要
- 特許の機構は Tech Rider が15年間量産で実証済
- 3 sliding interface のみ → 14ボール関節と比べて QC失敗率激減
- BOM 大幅減（弾性ボディ1 + 硬質四肢4 + 硬質swivel3）→ FOB USD 15 余裕で到達
- 特許失効済（2017-05-17 US）→ ロイヤリティ不要

### 旧 14関節 / Dummy-13 案について

旧案のリファレンス画像 `public/models/joint_reference.jpeg` は記録として残すが、VIMO Classic の仕様では **無効**。工場側に Dummy-13 として既に送付済の場合は **VIMO Classic の patent-based 仕様で上書き連絡が必要**（[[project_vimo_factory_rfq_comparison]] 参照）。

## 本体材質（2026-05-22 VIMO Classic 確定）

- **胴体**：**弾性エラストマー**（シリコン or TPE）。patent claim 通り resilient material。LABUBU 的触感価値を維持しつつ patent 構造で機械的安定。
- **四肢**：硬質プラスチック（PA / POM / PVC）。patent Fig.6-12 の snap-fit pin-and-socket をそのまま採用。
- **首 / 手首 swivel**：硬質プラ peg-in-socket（patent の関節と同じ家族の成形）。弾性腱（item 48）は省略＝拘束なし自由回転。

## 衣装の競技別テクスチャ差別化（2026-05-19 フランツ）

LABUBUの毛皮的「触り心地」を競技ユニフォームの素材差で表現:

| 競技 | 想定布素材 | 触感方向 |
|---|---|---|
| Snowboard | パファーナイロン＋カーゴコットン | しっとり＋ふかふか |
| Baseball | メッシュ＋ピンストライプ綿 | サラッ＋ザラッ |
| Golf | ポロ生地＋スラックスウール | スムース＋しっとり |
| Soccer | ドライメッシュ＋ショーツ綿 | サラサラ |
| Surf | ラッシュガード（ナイロン/エラスタン）＋ボードショーツ | つるん |

- 各競技で布の触感を変えることでブランドの「アスリート文化体験」を物理的に伝える
- コスト次第で全競技採用 or 段階展開を判断
